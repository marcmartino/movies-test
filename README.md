# Movies Evan Likes!

## Setup

1. Clone this repo.
2. `npm i` or `yarn` to install dependencies
3. `npm run dev` or `yarn dev` to start both the API and the React App

You should now have the `API` running on port 6001 and the `UI` running on port 3000
